Gitlet

十月 24, 2023 发布在 Linux, Git

基于java的命令行版本控制软件。

初始化

当前工作目录。

初始化存储库，在工作目录下创建.gitlet存储库目录。

其中包含第一次初始化提交。

add

添加文件，会向暂存区中添加文件名到Blob对象的映射。

然后将暂存区对象序列化到.gitlet/StagingArea文件。

此时Blob还没有真正添加到存储库。

Commit

保存快照，会创建一个Commit对象。

修改a.txt，删除b.txt，添加c.txt，创建新快照。

存储库。

branch

创建新分支，指向当前Commit。
