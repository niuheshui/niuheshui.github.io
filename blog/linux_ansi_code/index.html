ANSI控制字符

八月 5, 2023

什么是ANSI控制字符

ANSI控制字符是一组特殊的字符序列，用于在文本终端上执行各种控制操作。这些字符以Escape字符（ASCII码为27）开头，并由一系列特定的字符组成。

简单来说就是控制linux终端上的字符显示

格式

\033[<参数1>;<参数2>;<参数3>m

其中, \033表示Escape字符的转义序列, [表示参数序列的开始，m表示控制字符的结束.

参数可以是一个或多个以分号分隔的整数值，用于指定不同的控制选项, 这些参数的含义取决于具体的控制字符.

常见的参数及其含义:
指定文本样式或颜色的选项.
指定文本颜色或背景颜色的选项.
指定其他选项，如光标位置等.

设置前景色：
设置为默认颜色：\033[39m
设置为黑色：\033[30m
设置为红色：\033[31m
设置为绿色：\033[32m
设置为黄色：\033[33m
设置为蓝色：\033[34m
设置为洋红色：\033[35m
设置为青色：\033[36m
设置为白色：\033[37m
设置为256色中的某个颜色：\033[38;5;{color_code}m

设置背景色：
设置为默认颜色：\033[49m
设置为黑色：\033[40m
设置为红色：\033[41m
设置为绿色：\033[42m
设置为黄色：\033[43m
设置为蓝色：\033[44m
设置为洋红色：\033[45m
设置为青色：\033[46m
设置为白色：\033[47m
设置为256色中的某个颜色：\033[48;5;{color_code}m

在上述示例中，{color_code}是一个0到255之间的整数，表示颜色的索引。对于256色模式，可以使用0到255的索引来选择不同的颜色。

请注意，不同的终端可能对颜色的支持程度有所不同。某些终端可能只支持基本的8种颜色，而其他终端可能支持更多的颜色选项。因此，在使用ANSI控制字符设置颜色时，最好根据目标终端的规范或文档来选择适当的颜色选项。
