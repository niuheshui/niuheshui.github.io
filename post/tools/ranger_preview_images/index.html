ranger+kitty开启图像预览

五月 9, 2024

参考官方 wiki

创建配置文件副本

$ ranger --copy-config=rc

修改配置

set preview_images true
set preview_images_method kitty

配置文件中注释写到

Preview images in full color using kitty image protocol. Requires python PIL or pillow library. If ranger does not share the local filesystem with kitty the transfer method is changed to encode the whole image; while slower, this allows remote previews, for example during an ssh session. Tmux is unsupported.

安装 pillow 库

$ pip install pillow

! 不要修改 kitty 的 TERM 环境变量, 如果环境变量的 TERM 值不等于 kitty 就不会显示图片

# kitty/kitty.conf
term kitty

kitty 显示图片

$ kitten icat img-file

更多使用方法参考官方文档
