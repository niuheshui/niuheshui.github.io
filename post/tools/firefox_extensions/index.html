FireFox扩展受限

五月 8, 2024

解决 FireFox 扩展在某些网页不能使用问题

禁用保护

在地址栏里输入 about:config 然后按回车

有时会出现警告页面, 点击接受风险并继续以打开 about:config 页面

搜索 extensions.quarantinedDomains.enabled 首选项

如果该选项不存在, 则会显示添加选项, 选择 boolean 类型, 点击 + 按钮创建, 将其值更改为 false

重启 Firefox

移除受限域名

地址栏输入 about:config

添加新的 boolean 选项 privacy.resistfingerprinting.block.mozaddonmanager, 具有 TRUE 值

在 extensions.webextensions.restrictedDomains 选项中移除受限域名
