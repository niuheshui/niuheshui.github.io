nmcli管理网络连接

二月 25, 2024

nmcli 是用于控制 NetworkManager 的命令行工具。NetworkManager 是一个系统网络服务，用于管理各种类型的网络连接，包括以太网、Wi-Fi、VPN 等。nmcli 允许用户通过命令行界面管理 NetworkManager 管理的网络连接，包括查看、创建、编辑、激活和停用连接。

使用示例

开启/关闭wifi

$ nmcli radio wifi on/off

显示可用wifi

$ nmcli dev wifi list
*       62:6D:04:BA:E8:AB  iPhone 15 Pro Max             Infra  149   270 Mbit/s  100     ▂▄▆█  WPA2 WPA3

连接wifi

$ nmcli dev wifi connect "iPhone 15 Pro Max" password 12345678

显示连接

$ nmcli connection show
NAME                 UUID                                  TYPE      DEVICE  
iPhone 15 Pro Max    19b4d43c-197a-4bbe-89ed-8db4b2f99201  wifi      wlp4s0  
docker0              4928a251-fd80-41b2-9ee0-130dcb72340c  bridge    docker0

显示所有设备的状态

$ nmcli device status
DEVICE          TYPE      STATE         CONNECTION        
wlp4s0          wifi      已连接        iPhone 15 Pro Max 
docker0         bridge    连接（外部）  docker0           
p2p-dev-wlp4s0  wifi-p2p  已断开        --                
eno1            ethernet  不可用        --                
lo              loopback  未托管        --
