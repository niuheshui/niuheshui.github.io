在 Makefile 中为变量替换空格

五月 11, 2024

将一个以空格分隔的变量使用字符串拼接

不需要废话

# Makefile
EMPTY := 
SPACE := $(EMPTY) $(EMPTY)

variable = a b c
# a:b:c
$(info $(subst $(SPACE),:,$(variable)))

GNU Make 字符串函数文档: https://www.gnu.org/software/make/manual/html_node/Text-Functions.html

以及万能的 StackOverflow: https://stackoverflow.com/questions/7748460/add-string-in-between-variables-for-gnu-make
