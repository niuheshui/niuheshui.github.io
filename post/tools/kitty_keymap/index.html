五月 10, 2024

详细配置 https://github.com/niuheshui/dotfiles/blob/main/kitty/kitty.conf

Install

https://sw.kovidgoyal.net/kitty/binary/

安装最新版本

$ curl -L https://sw.kovidgoyal.net/kitty/installer.sh | sh /dev/stdin \
    installer=nightly

自定义安装位置

$ curl -L https://sw.kovidgoyal.net/kitty/installer.sh | sh /dev/stdin \
    dest=/some/other/location

Themes

https://sw.kovidgoyal.net/kitty/kittens/themes/

$ kitten themes
# or
$ kitty +kitten themes
