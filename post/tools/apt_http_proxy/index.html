四月 16, 2024

为apt设置代理

临时方案

$ sudo apt -o Acquire::http::proxy="http://:7890/" install ...

永久方案

$ cat /etc/apt/apt.conf
Acquire::http::proxy "http://127.0.0.1:7890/";
