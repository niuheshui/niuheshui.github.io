2. ipconfig

ubuntu22.04 中有关 DNS 信息

  1. 查看 /etc/resolv.conf 文件：

    打开终端并输入以下命令：

    cat /etc/resolv.conf

    在输出中，你会看到一行或多行以 “nameserver” 开头的条目，每个条目后面跟着一个 IP 地址。这些 IP 地址就是当前系统配置的 DNS 服务器。

  2. 使用 nmcli 命令：

    如果你的系统使用 NetworkManager 来管理网络连接，你可以使用 nmcli 命令来查看网络配置信息，包括 DNS 服务器的 IP 地址。在终端中输入以下命令：

    nmcli dev show | grep 'IP4.DNS'

    这将显示当前网络设备配置的 IPv4 DNS 服务器的 IP 地址。

  3. 使用 resolvectl 命令：

    如果你的系统使用 systemd-resolved 来管理 DNS 解析，你可以使用 resolvectl 命令来查看 DNS 配置信息。在终端中输入以下命令：

    resolvectl status | grep 'DNS Servers'

    这将显示当前系统使用的 DNS 服务器的 IP 地址。
